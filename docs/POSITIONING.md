# Positioning

Wishwright is for indie developers who want product ideas grounded in public requests but do not
want to spend hours searching posts and judging each one by hand. It turns request-shaped posts
into a ranked, safety-filtered shortlist and an auditable build brief, so a builder can start from
evidence instead of a blank page.

## Product

- **Display name:** Wishwright
- **Tagline:** Turn public wishes into safer build briefs
- **Voice:** Direct, technical, candid about integration boundaries, and specific about outputs
